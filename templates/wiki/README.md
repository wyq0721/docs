# Wiki 母本模板

这套模板用于沉淀 AI 实践相关的 wiki 母本，方便后续基于真实素材快速制作正式内容。

## 建议使用方式

1. 先复制本目录结构到你的实际工作目录。
2. 优先填写：
   - `01-项目总览.md`
   - `03-典型案例库.md`
   - `04-方法论沉淀.md`
3. 再根据素材补充时间线、效果、风险与对外表达部分。

## 推荐目录结构

```text
01-项目总览
02-推进时间线
03-典型案例库
04-方法论沉淀
05-效果与价值证明
06-风险与治理
07-对外表达素材库
08-附录
```

## 文件说明

- `01-项目总览.md`：说明为什么做、做什么、当前整体状态。
- `02-推进时间线.md`：记录从启动到当前阶段的推进过程。
- `03-典型案例库.md`：沉淀可复用案例。
- `04-方法论沉淀.md`：总结场景选择与落地方式。
- `05-效果与价值证明.md`：整理效率、质量、组织价值证据。
- `06-风险与治理.md`：明确边界、审核与治理要求。
- `07-对外表达素材库.md`：为 PPT、汇报、对外介绍准备素材。
- `08-附录.md`：归档工具、模板、FAQ 与资料索引。

## 使用原则

- wiki 写完整，PPT 从 wiki 抽取。
- 优先写事实，再写判断，最后写对外表达。
- 所有对外内容都应基于人工复核后的信息。
